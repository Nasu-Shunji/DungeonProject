# Dungeon Project

この世界は、AIと人が共に歴史を作るダンジョンである。

Unityで制作した、小規模な3Dダンジョン探索ゲームです。

ダンジョン内を探索し、扉を開け、宝箱からアイテムを入手して出口を目指します。

## Gameplay

1. ダンジョン内を探索する
2. 扉を開ける
3. 宝箱からアイテムを取得する
4. 出口に到達する

## Controls

| 操作 | キー |
|---|---|
| 移動 | WASD / 矢印キー |
| インタラクト | E |
| カメラ回転 | 右クリックを押しながらマウス移動 |
| ポーズ | Esc |

## Features

- カメラ基準のPlayer移動
- 右クリック中の三人称カメラ回転
- カメラの壁めり込み防止
- 扉の開閉
- 宝箱とアイテム取得
- アイテム数UI
- ダンジョンクリア判定
- 落下時のリスポーン
- ポーズ、タイトル復帰、ゲーム終了
- BGMと効果音
- Windows向けビルド

## Development Environment

- Unity 6
- Universal Render Pipeline
- C#

## How to Play

Windows版のフォルダを展開し、`DungeonProject.exe`を実行してください。

`.exe`だけを別の場所へ移動すると起動できないため、同梱されているフォルダやDLLと一緒に保管してください。
